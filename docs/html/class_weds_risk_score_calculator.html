Detailed Description

Manages the execution and state of the TinyML fire risk prediction model.

This class maintains a 7-day rolling history of environmental data to compute the required time-series features (averages and deltas) before feeding them into the quantized TensorFlow Lite neural network. The rolling history is stored in RTC-retained memory so it survives hardware deep-sleep resets.
